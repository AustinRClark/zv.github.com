Artifact is an nonrelational database written in Erlang, based on Dynamo.

Latest Version: 0.2

Interested?

  • Download latest release (0.2)
  • Read the documentation or download as
  • Fork it on github
  • Add issues and feature requests

What's the elevator pitch?

  • Load balancing and reliability - No Single point of failure
  • High scalability and elasticity for low operation cost
  • Automatic Load Balacing
  • Low Latency
  • Uses Memcached Protocol
  • Fast Converging Eventual Consistancy
  • Temporally indepedent + versioning
